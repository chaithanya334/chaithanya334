### 🦠 COVID-19 Patient Severity Classification

![Python](https://img.shields.io/badge/Python-FFD43B?style=flat-square&logo=python&logoColor=blue)
![Scikit-Learn](https://img.shields.io/badge/scikit--learn-F7931E?style=flat-square&logo=scikit-learn&logoColor=white)
![Pandas](https://img.shields.io/badge/Pandas-150458?style=flat-square&logo=pandas&logoColor=white)
![Tkinter](https://img.shields.io/badge/Tkinter-GUI-00AFF0?style=flat-square)
![Matplotlib](https://img.shields.io/badge/Matplotlib-FF6B6B?style=flat-square&logo=python&logoColor=white)

</div>

> 🏥 A desktop GUI application that classifies the **severity of COVID-19 infection** based on clinical parameters — helping clinicians prioritize patient care faster.

| Parameter | Description |
|-----------|-------------|
| 🩸 **SpO2** | Blood oxygen saturation level |
| 🫁 **Respiratory Rate** | Breaths per minute |
| 🔬 **CRP (mg/L)** | C-Reactive Protein inflammation marker |

**Output:** Classifies patient condition as 🟢 **Mild** / 🟡 **Moderate** / 🔴 **Severe**

> 🎓 *Master's Final Year Project — KSWU, Vijayapura*
